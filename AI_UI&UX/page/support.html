Getting Started

Everything you need to know to get up and running with Isabel AI.

What is Isabel AI and what can it do? +

Isabel AI is a conversational AI assistant designed to help users with a wide range of tasks, including answering questions, providing information, and assisting with various digital activities. It supports voice input and output, making it accessible for users who prefer spoken interaction. Isabel AI can assist with tasks such as setting reminders, managing schedules, and providing information on various topics.

How do I create an account? +

To create an account, click on the "Sign Up" button in the top right corner of the website. You will be prompted to enter your email address and create a password. After verifying your email, you can log in and start using Isabel AI.

Do I need to install anything? +

No, Isabel AI is a web-based application that can be accessed directly through your browser. There is no need to download or install any software. Simply visit the website and log in to start using the service.

Is Isabel AI free to use? +

Isabel AI offers both free and premium plans. The free plan includes basic features and limited usage, while the premium plan offers additional features and higher usage limits. You can choose the plan that best suits your needs.

Which browsers are supported? +

Isabel AI supports the latest versions of popular web browsers, including Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge. We recommend using the latest version of your preferred browser for the best experience.
